📋 1. Parterna

Detta avtal gäller mellan Spick Sverige AB (org.nr 559402-4522) och dig som registrerad städare på plattformen spick.se.

✅ Du arbetar som egenföretagare, inte anställd hos Spick. Spick är en förmedlingsplattform.

🏢 2. Krav för att vara städare

  • Registrerad enskild firma eller aktiebolag hos Bolagsverket
  • Giltig F-skattsedel från Skatteverket
  • Giltig ansvarsförsäkring (minst 1 Mkr)
  • Godkänd identitetskontroll via BankID
  • Minst 18 år gammal

⚠️ Utan giltig F-skattsedel kan Spick inte betala ut ersättning och ditt konto pausas.

🎁 4. Bonusprogram

  • Brons (standard): 290 kr/h – ingångsnivå
  • Silver (4.7★+, 20+ jobb): +5% bonus = ~305 kr/h
  • Guld (4.8★+, 50+ jobb): +10% bonus = ~319 kr/h
  • Platinum (4.9★+, 100+ jobb): +15% bonus = ~334 kr/h

✅ Bonusnivå beräknas automatiskt varje måndag och justeras retroaktivt.

⭐ 5. Kvalitetskrav

  • Utför uppdrag professionellt och enligt kundens instruktioner
  • Var alltid punktlig – kontakta kund om du är försenad
  • Meddela Spick minst 24 timmar i förväg om du måste avboka
  • Behandla kundens hem och egendom varsamt
  • Snittbetyg under 4.0 kan leda till kontopaus

🛡️ 6. Nöjdhetsgaranti

Spick erbjuder kunder 100% nöjdhetsgaranti. Om en kund ger betyg 1–2 stjärnor:

  • Spick kontaktar dig för att förstå vad som hände
  • En gratis omstädning kan beordras inom 48h
  • Vid omstädning ersätts du för din tid som vanligt

⚠️ Upprepade garantiärenden kan leda till kontopaus.

🔒 7. Konfidentialitet

  • Du hanterar kundinformation konfidentiellt
  • Dela inte kunders adresser, telefonnummer eller personuppgifter
  • All kundkontakt sker via Spicks plattform
  • Du får inte kontakta kunder direkt för att erbjuda tjänster utanför Spick

❌ 8. Uppsägning

  • Du kan avsluta samarbetet när som helst via hello@spick.se
  • Spick kan avsluta samarbetet med 14 dagars varsel
  • Vid allvarliga brott mot villkoren kan kontot stängas omedelbart
  • Utestående betalningar regleras oavsett anledning till avslut

Admin-guide

📅 Daglig rutin (5 min)

1
Kolla Dashboard
Finns det röda varningar? Öppna garantier? Nya bokningar som behöver åtgärd?
2
Bokningsbekräftelse
Alla nya bokningar får automatisk bekräftelse via mail. Du behöver bara kontrollera att statusen ändras till "bekräftad".
3
Betygsmail
Klicka "Skicka betygsmail" under Bokningar för att skicka betygsförfrågan till kunder som städat igår.

👤 Godkänna ny städare

1
Gå till Ansökningar
Se alla inkomna ansökningar. Klicka "Godkänn" – städaren läggs automatiskt till på stadare.html.
2
Välkomstmail skickas automatiskt
Städaren får automatiskt ett välkomstmail med onboarding-instruktioner.
3
Fyll i email i Städare-fliken
VIKTIGT: Lägg in städarens e-postadress så de kan ta emot bokningsnotiser.

🛡️ Nöjdhetsgaranti

1
Automatisk trigger vid ≤2★
Du får automatisk notis. Garantiärendet syns under "Garantier" med status "open".
2
Kontakta kunden via mail
Klicka "Kontakta" för att öppna ett förifyllt mail till kunden med erbjudande om kompletterande städning.
3
Vem betalar?
Städaren utför returbesöket utan extra kostnad – det ingår i partnersavtalet. Spick betalar ingen extra provision. Triggas mer än 2 gånger/månad → varning till städaren.

💰 Månadsvis provision

1
Sista dagen varje månad
Klicka "Generera fakturor" under Fakturor – systemet beräknar automatiskt 17% provision per städare.
2
Fakturan skickas automatiskt
Varje städare får ett mail med fakturabelopp och Swish-instruktioner (betalas till Spick AB).

💾 Backup & säkerhet

1
Supabase automatisk backup
Supabase Free-plan har 7 dagars Point-in-Time-Recovery. Uppgradera till Pro (25$/mån) för 30 dagars backup.
2
GitHub versionshistorik
All kod är versionshanterad på GitHub. Rulla tillbaka till valfri version om något går fel.
3
Vid hack/intrång
1) Rotera alla API-nycklar omedelbart. 2) Kontakta Supabase support. 3) GitHub: Settings → Secrets → uppdatera. 4) Resend: dashboard.resend.com → revoke key.
